AI-Based Agricultural Loan Credit Scorer
📌 Overview
The AI-Based Agricultural Loan Credit Scorer predicts farmers’ creditworthiness using non-traditional agricultural data such as crop history, soil health, and weather patterns. It generates a Trust Score to help lenders make fair, data-driven loan decisions.

🚀 Problem Statement
Over 70% of smallholder farmers lack formal credit history, making them invisible to banks. Traditional credit scoring systems fail to evaluate their real repayment capacity.

💡 Solution
Our system uses Machine Learning (XGBoost) to analyze alternative agricultural data and predict default risk. The model converts risk probability into a simplified Trust Score (0–100) for lenders.
